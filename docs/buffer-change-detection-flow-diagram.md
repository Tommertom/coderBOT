## Key Points

### 1. Initialization
- Callback bound when session is created
- Monitoring starts immediately after session creation

### 2. Monitoring Loop
- Runs every 1000ms (1 second)
- Compares current buffer with last snapshot
- Tracks time since last change

### 3. Detection Logic
```typescript
if (currentBuffer !== lastBufferSnapshot) {
    // Buffer changed - reset timer
    lastBufferSnapshot = currentBuffer;
    lastBufferChangeTime = now;
} else {
    // Buffer unchanged - check elapsed time
    timeSinceLastChange = now - lastBufferChangeTime;
    if (timeSinceLastChange >= 5000) {
        // Trigger notification!
    }
}
```

### 4. Cleanup
- Interval cleared after notification
- Prevents duplicate notifications
- Resources freed properly
